# Tunnel-management-system

## 学习vue3所练习的项目

## 项目所需技术栈

1. 基本框架：vue3
2. 网络请求：axios
3. 路由：Vue-Router4.x
4. 状态管理：Pinia
5. 图表：echarts
6. 语言切换：vue-i18n
7. UI组件库：element-plus
8. 仓库持久化存储: `cnpm i --save pinia-plugin-persist`
9. ElementPlus字体图标: @element-plus/icons-vue

## 跨域的解决方案

1. 前台
    1. jsonp
    2. Proxy(开发环境生效)
2. 后台解决
    1. cors解决跨域
